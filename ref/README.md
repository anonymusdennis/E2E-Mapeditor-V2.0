# Game reference (decompiled)

Decompiled C# source from **The Escapists 2** (`Assembly-CSharp.dll` and
`Assembly-CSharp-firstpass.dll`), produced with ilspycmd for mod development
reference only.

| Path | Contents |
|------|----------|
| `decompiled/Assembly-CSharp/` | Main game assembly (~3,700 classes) |
| `decompiled/Assembly-CSharp-firstpass/` | First-pass assembly (Steamworks, Nex, etc.) |

This is **not** a runnable project — use it to look up types, method names, and
editor internals while working on E2EApi / MapEditorMod. The live working copy
during development stays in gitignored `reference/` (same layout).

**Do not redistribute** as a substitute for owning the game.
